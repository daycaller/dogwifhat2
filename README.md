## Local development

Just open `index.html` in a browser. There is no build step.

For a local server (recommended for testing the Web Share API since it requires a secure context):

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy

Connected to Vercel via GitHub. Push to `main` → auto-deploys.
